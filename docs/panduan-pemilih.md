# Panduan Pemilih E-Pilketos

Pengguna: Siswa / Pemilih

---

## Apa itu E-Pilketos?

E-Pilketos adalah sistem pemilihan Ketua dan Wakil Ketua OSIS secara online.

- Setiap siswa mendapat satu token dari panitia.
- Token digunakan untuk masuk ke halaman pemilihan.
- Token hanya bisa dipakai sekali untuk memilih.
- Identitas (NIS/nama/kelas) tidak dimasukkan ke sistem.

---

## Apa itu Token?

- Token adalah kode unik, contoh: `7RNA-25LK`.
- Token diberikan oleh panitia.
- Token hanya bisa dipakai satu kali untuk memberikan suara.
- Token tidak boleh dibagikan ke orang lain.

Jika token hilang atau rusak, hubungi panitia.

---

## Langkah-Langkah Voting

### 1) Buka halaman pemilihan

1. Buka browser di HP atau komputer.
2. Masukkan alamat website E-Pilketos dari panitia.
3. Buka halaman voting:
   `/vote`

### 2) Masukkan token

1. Ketik token sesuai yang tertulis di kartu token.
2. Tekan tombol lanjut.

Token hanya valid jika:

- belum pernah dipakai
- pemilihan sedang berlangsung (status `ACTIVE`)
- masih dalam jadwal pemilihan

Jika token ditolak, minta bantuan panitia.

### 3) Pilih pasangan calon

1. Setelah masuk, akan muncul daftar pasangan calon.
2. Baca informasi kandidat (nama, kelas, visi, misi, program).
3. Pilih satu pasangan calon.
4. Konfirmasi pilihan.
5. Kirim suara.

### 4) Setelah memilih

- Suara tercatat.
- Token langsung habis dan tidak bisa dipakai ulang.
- Pilihan tidak bisa diubah.

---

## Melihat Hasil

Halaman hasil publik:

- `/results`

Hasil hanya bisa dilihat jika panitia sudah mempublikasikan hasil. Jika belum, halaman akan menampilkan bahwa hasil belum dipublikasikan.

---

## Hal yang Perlu Dijaga oleh Pemilih

- Simpan token baik-baik sebelum digunakan.
- Jangan memperlihatkan token ke orang lain.
- Jangan memotret token dan menyebarkannya.
- Gunakan hak suara dengan jujur.

---

## Jika Terjadi Masalah

Hubungi panitia jika:

- Token tidak diterima.
- Muncul pesan token sudah digunakan padahal belum memilih.
- Halaman tidak bisa diakses.
- Terjadi error saat konfirmasi atau kirim suara.
