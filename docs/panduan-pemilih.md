# Panduan Pemilih E-Pilketos

Pengguna: Siswa / Pemilih

---

## Apa itu E-Pilketos?

E-Pilketos adalah sistem pemilihan Ketua dan Wakil Ketua OSIS yang dilakukan secara online.

- Setiap siswa mendapat satu **token** dari panitia.
- Token digunakan untuk masuk ke halaman pemilihan.
- Token hanya bisa dipakai **sekali** untuk memilih.
- Nama dan NIS tidak dimasukkan ke sistem, sehingga pilihan tetap rahasia.

---

## Apa itu Token?

- Token adalah kode unik, misalnya: `7RNA-25LK`.
- Token diberikan oleh panitia secara langsung.
- Token hanya bisa dipakai satu kali untuk memberikan suara.
- Token tidak boleh dibagikan ke orang lain.

Jika token hilang atau rusak, segera hubungi panitia di sekolah.

---

## Langkah-Langkah Voting

### Membuka halaman pemilihan

1. Buka browser di HP atau komputer.
2. Masukkan alamat website E-Pilketos yang diberikan oleh panitia.
3. Di halaman utama, pilih menu **Mulai Memilih** (atau menu serupa yang disediakan).

### Mengisi token

1. Di halaman voting, akan muncul form untuk memasukkan token.
2. Ketik token sesuai yang tertulis di kartu token.
3. Tekan tombol untuk melanjutkan.

Jika token benar dan masih berlaku, Anda akan masuk ke halaman daftar calon.  
Jika token ditolak, minta bantuan panitia.

### Memilih pasangan calon

1. Halaman akan menampilkan daftar pasangan calon Ketua & Wakil Ketua OSIS.
2. Baca informasi yang tersedia:
   - Nama ketua dan wakil.
   - Kelas.
   - Visi, misi, dan program kerja.
3. Pilih satu pasangan calon yang menurut Anda paling sesuai.
4. Tekan tombol pilih pada pasangan calon tersebut.
5. Sistem akan menampilkan konfirmasi, pastikan pilihan sudah benar.
6. Tekan setuju/lanjut untuk mengirim suara.

### Setelah memilih

1. Setelah suara terkirim, token langsung dianggap habis (tidak bisa dipakai lagi).
2. Anda akan melihat halaman bahwa suara sudah tercatat.
3. Setelah itu, Anda tidak bisa mengganti pilihan.

---

## Hal yang Perlu Dijaga oleh Pemilih

- Simpan token baik-baik sebelum digunakan.
- Jangan memperlihatkan token kepada teman atau orang lain.
- Jangan memotret token dan menyebarkannya di media sosial.
- Jangan mencoba menggunakan token milik orang lain.
- Gunakan hak suara dengan jujur dan bertanggung jawab.

---

## Jika Terjadi Masalah

Segera hubungi panitia jika:

- Token tidak diterima oleh sistem.
- Muncul pesan bahwa token sudah digunakan, padahal Anda belum memilih.
- Halaman pemilihan tidak bisa diakses.
- Terjadi error saat Anda menekan tombol pilih.

Panitia akan memeriksa data di sistem dan memberi solusi (misalnya pergantian token atau pengecekan teknis).

---

## Catatan

- Sistem tidak menyimpan nama, NIS, atau kelas Anda di dalam database.
- Suara Anda hanya dicatat sebagai pilihan terhadap salah satu pasangan calon.
- Kerahasiaan pilihan dijaga melalui penggunaan token anonim.
